# Лабиринт Сокровищ

Текстовая приключенческая игра в жанре interactive fiction. Игрок исследует таинственный лабиринт, решает загадки, собирает предметы и ищет спрятанное сокровище.


## Структура проекта:

labyrinth_game/
├── main.py           # Точка входа и игровой цикл
├── constants.py      # Данные игры (комнаты, команды)
├── player_actions.py # Действия игрока
└── utils.py          # Вспомогательные функции


## Установка:

    make install      # Установка зависимостей
    
    make lint         # Проверка стиля кода
    
    make build        # Сборка пакета
    
    make publish      # Тестовая публикация
    
    make clean        # Очистка временных файлов


## Технологии

    Python 3.8+

    Poetry 1.8.0+

    Ruff (для проверки кода)


# Игровой процесс
### Основные команды:

    north/south/east/west - перемещение между комнатами

    look - осмотреть текущую комнату

    take <предмет> - взять предмет

    inventory - показать инвентарь

    use <предмет> - использовать предмет

    solve - решить загадку в комнате

    help - показать справку по командам

    quit - выйти из игры

### Особенности игры:

    6 уникальных комнат с разными тематиками

    Интерактивные загадки с альтернативными ответами

    Система предметов с уникальными взаимодействиями

    Случайные события при перемещении между комнатами

    Ловушки которые могут активироваться

    Условие победы: найти и открыть сундук с сокровищами

### Комнаты лабиринта:

    Вход (entrance) - начальная комната с факелом

    Зал (hall) - центральная комната с первой загадкой

    Комната с ловушкой (trap_room) - опасная комната с ключом

    Библиотека (library) - комната с древними свитками

    Оружейная (armory) - комната с оружием и шкатулкой

    Сокровищница (treasure_room) - финальная комната с сундуком
